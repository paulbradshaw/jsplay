A responsive page

This page uses the Bootstrap JavaScript library for responsive webpages. Specifically, this page is using the Jumbotron theme which is applied by putting most of the page in a div class="jumbotron". Perhaps this section (in class="container") has your biggest story. On a desktop it runs across the whole page. At the bottom is a button which is generated by the a class="btn btn-primary btn-lg"
